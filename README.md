# React Boilerplate

## Features

* [x] [Express](http://expressjs.com/) compressed static server
* [x] [Jade](http://jade-lang.com/) template engine build with
  [Gulp](http://gulpjs.com/)

## Getting Started

### Installation

```shell
npm install
```

### Building

Development:

```shell
gulp
```

Production:

```shell
gulp build
```

### Running Server

```shell
npm start
```
